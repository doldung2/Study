# 01. p와 y개수 구하기
### java
'''
class Solution {
    boolean solution(String s) {
        boolean answer = true;
        
        String text = s.toUpperCase();
        
        int pCnt = 0;
        int yCnt = 0;
        
        for(int i = 0; i<s.length(); i++){
            if(text.charAt(i)=='P'){
                pCnt++;
            }else if(text.charAt(i)=='Y'){
                yCnt++;
            }else{
                //아무값을 설정안하므로 pCnt,yCnt는 모두 0이다.
            }            
        }
                
        if(pCnt == yCnt){
            answer = true;
        }else{
            answer = false;
        }
        
        return answer;
    }
}
'''
